# PdM Skills Builder 🧠

**119 готовых к работе скиллов для продакт-менеджеров**, упакованных в удобный визуальный конструктор промптов.

Никакой воды. Никаких шаблонных "супер-промптов на 100 страниц", которые сбивают контекст нейросети. Это модульный конструктор, где вы выбираете только нужные вам продуктовые навыки и за один клик собираете идеальный системный промпт (или файл `.cursorrules` / `.clinerules`) для любой LLM (Cursor, Antigravity, Claude, ChatGPT, Windsurf).

## 🔥 Фичи

* **Модульность:** 119 изолированных навыков. От дизайна A/B-тестов до SaaS-метрик, расчета Unit-экономики и B2B закупочных комитетов.
* **Готовые комбо-сборки:** Не хотите собирать вручную? Используйте готовые пресеты под роли (Fintech PM, E-commerce PM, Growth, CPO).
* **Жесткие Guardrails:** Скиллы заставляют ИИ работать по жестким правилам. Например, единое "правило 5 вопросов" для любой метрики, никаких наводящих вопросов в кастдевах.
* **100% Локально и безопасно:** Инструмент работает прямо в вашем браузере. Никаких бэкендов, баз данных и серверов. 

## 🚀 Как использовать

Инструмент не требует установки серверов или баз данных. Он работает прямо в браузере!

### Вариант 1: Использовать онлайн (Рекомендуется)
Просто откройте онлайн-версию конструктора:
👉 **[Запустить PdM Skills Builder](https://anton-creates.github.io/pdm-skills-builder/)**

### Вариант 2: Запустить локально
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Anton-Creates/pdm-skills-builder.git
   ```
2. Откройте файл `index.html` в любом браузере.
3. Готово!

## 🛠️ Рабочий процесс (Как применять промпты)

1. Откройте конструктор в браузере.
2. Выберите категорию (например, `Финтех` или `Аналитика`) или готовую Комбо-сборку.
3. Накликайте навыки, которые вам нужны для текущей задачи (рекомендуем 3-4 скилла за раз, чтобы не перегружать контекст ИИ).
4. Нажмите кнопку **"Скопировать сборку"** (внизу слева).
5. Вставьте скопированный текст в ваш Claude, ChatGPT или сохраните как `.cursorrules` в вашем рабочем проекте в Cursor.

## 🤝 Как добавить свой скилл

Хотите расширить базу? Добавление нового скилла занимает минуту!
1. Создайте новую папку в директории `skills/` (например, `skills/my-new-skill`).
2. Создайте внутри файл `SKILL.md` (посмотрите структуру соседних скиллов).
3. Запустите скрипт `python manager.py` (или `run_manager.bat` на Windows), чтобы пересобрать базу данных `skills_db.js`.
4. Обновите страницу в браузере — ваш скилл появится в интерфейсе!
