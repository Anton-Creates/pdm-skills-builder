# B2B-телеком продукты (b2b-telecom)

Разработать требования к корпоративным телеком-продуктам (виртуальная АТС, облачный колл-центр, VPN для бизнеса) и их интеграции с CRM-системами клиентов.

## Процесс
1. **Опиши сценарии интеграции с CRM.** Автоматическое открытие карточки клиента в CRM при входящем звонке на виртуальную АТС, запись разговоров.
2. **Определи параметры SLA.** Доступность каналов связи (например, 99.99%), резервные каналы.
3. **Спроектируй личный кабинет корпоративного клиента.** Управление лимитами сотрудников, заказ SIM-карт.

## Формат вывода
```
## Спецификация B2B телеком-продукта: [Название]
- **Сценарий интеграции:** API-вебхуки для передачи записи звонков в Битрикс24 / amoCRM.
- **SLA доступности телефонии:** 99.99% (не более 4 минут простоя в месяц).
- **Личный кабинет:** управление бюджетами SIM-карт сотрудников с лимитами трат в месяц.
```
