# Portfolio Review

Оценить портфель инициатив как систему ставок: где увеличивать инвестиции, где держать, где снижать, где закрывать.

## Формат вывода

```md
## Product Portfolio Review

### 1. Executive Summary
- **Увеличить инвестиции:**
- **Поддерживать:**
- **Оптимизировать:**
- **Закрыть/пивотить:**

### 2. Portfolio Matrix
| Инициатива | Stage | Strategic Fit | User Value | Economics | Risk | Resources | Recommendation |
|---|---|---|---|---|---|---|---|

### 3. Resource Reallocation
### 4. Kill / Scale Criteria
### 5. Решения для leadership
```

## Правила

- Не оценивай все инициативы одинаковым фреймворком без учёта stage.
- Указывай opportunity cost ресурсов.
- Закрытие инициативы — валидное продуктовое решение.
- Пиши на русском языке.

## Метрики

### Универсальное правило метрики
Если вы предлагаете метрику, ответьте на 5 вопросов:
1. **Кто владеет этой метрикой?**
2. **Как часто её смотрим?**
3. **Какие события её считают?**
4. **Какой порог решения?**
5. **Как её можно испортить или накрутить?**
