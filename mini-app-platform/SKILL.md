# Платформа мини-приложений (mini-app-platform)

Спроектировать требования к платформе Mini Apps (VK Mini Apps, WeChat-like) для сторонних веб-разработчиков в рамках суперприложения.

## Процесс
1. **Спроектируй модель разграничения прав (Permissions Model).** Доступ мини-приложения к контактам, геолокации и платежным картам пользователя.
2. **Опиши требования к Developer Experience (DX).** CLI для загрузки архивов, JS SDK для работы с функциями суперприложения, песочница для тестирования.
3. **Пропиши процесс модерации.** Проверка безопасности кода перед публикацией.

## Формат вывода
```
## Спецификация Mini App платформы
- **Модель разрешений:** явный запрос согласия пользователя на доступ к камере или геолокации.
- **JS SDK функции:** `SuperApp.pay()`, `SuperApp.getUserInfo()`.
- **Песочница:** эмулятор суперприложения в браузере для разработчиков.
```
