# Корпоративное обучение B2B (b2b-edtech)

Разработать требования к LMS-системе (Learning Management System) для корпоративного сектора: обучение сотрудников, оценка навыков, интеграции.

## Процесс
1. **Опиши роли.** Администратор компании (HR), Руководитель, Сотрудник (студент).
2. **Спроектируй кабинет HR.** Назначение курсов сотрудникам, отслеживание прогресса, выгрузка отчетов о прохождении обязательного комплаенс-обучения.
3. **Определи бизнес-ROI для компании.** Снижение затрат на адаптацию сотрудников (onboarding time), рост навыков.

## Формат вывода
```
## Спецификация B2B EdTech системы: [Название]
- **Кабинет HR:** дашборд прогресса прохождения курсов по отделам.
- **Интеграция с ИТ-контуром:** импорт оргструктуры из 1С:ЗУП / Active Directory.
- **Комплаенс-модуль:** автоматическая блокировка доступа к рабочим системам при просрочке ежегодного теста по ИБ.
```
