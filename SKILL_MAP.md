## Быстрый выбор по ситуации

| Если нужно... | Используй |

| Задача | Скилл | Когда выбирать |

## Похожие скиллы: как не перепутать

| Если думаешь между... | Выбор |
|---|---|
| `/product-teardown` vs `/competitor-scan` | Teardown — один продукт глубоко; scan — рынок и конкуренты шире |
| `/competitor-scan` vs `/competitive-moat` | Scan — кто вокруг; moat — почему нас сложно скопировать/обойти |
| `/decision-doc` vs `/business-case` | Decision — одно решение на 1-2 страницы; business-case — крупная ставка с экономикой |
| `/prd` vs `/business-case` | PRD — что строим; business-case — зачем инвестируем и почему экономика сходится |
| `/ai-feature-spec` vs `/llm-product-design` | AI feature — любой ML/AI; LLM — генерация, RAG, prompts, eval, hallucinations |
| `/catalog-strategy` vs `/marketplace-catalog` | Catalog strategy — owned/e-com каталог; marketplace-catalog — seller-generated каталог |
| `/growth-loop` vs `/referral-mechanics` | Growth loop — вся петля роста; referral — конкретная механика приглашений |
| `/product-health-review` vs `/metrics-analyzer` | Health review — управленческая диагностика; metrics-analyzer — анализ данных/CSV |
